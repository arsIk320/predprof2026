# Управление столовой

Веб-приложение для автоматизации управления школьным питанием,
разработанное на **Flask + SQLite**.\
Проект заменяет бумажный учёт и упрощает взаимодействие между учениками,
поварами и администрацией.

------------------------------------------------------------------------

## Основные возможности

### Ученик

-   Просмотр меню завтраков и обедов
-   Оплата питания (разово или абонемент)
-   Подтверждение получения питания
-   Управление аллергенами и предпочтениями
-   Оставление отзывов
-   Получение уведомлений

### Повар

-   Учёт выданных блюд
-   Контроль остатков продуктов
-   Управление доступностью блюд
-   Создание заявок на закупку
-   Просмотр статистики и истории выдачи

### Администратор

-   Просмотр аналитики и статистики
-   Обработка заявок на закупку
-   Настройка абонементов
-   Формирование отчётов (PDF / CSV / JSON)
-   Отправка уведомлений

------------------------------------------------------------------------

## Docker

``` bash
cd predprof
docker-compose up --build
```

Открыть в браузере: **http://localhost:8080**

Данные сохраняются в: - `./data/canteen.db` - `./data/reports/`

------------------------------------------------------------------------

## Локальный запуск

``` bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Адрес по умолчанию: **http://localhost:8080**
